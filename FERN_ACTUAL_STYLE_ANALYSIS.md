## THE FERN PRODUCTION PIPELINE (ACTUAL)

Based on analysis, here's how Fern ACTUALLY makes their videos:

### STEP 1: SCRIPT & STORYBOARD
- Write narrative script
- Identify key visual moments
- Decide: 3D model, stock footage, or graphic?

### STEP 2: ASSET CREATION
**3D Models:**
- Create or license 3D models (planes, vehicles, buildings)
- Texture mapping for photorealism
- Rig for animation if needed

**Stock Footage:**
- License stock footage (clouds, landscapes, cities)
- Color grade to match
- Composite with 3D elements

**Photos:**
- Collect portrait photos
- Remove backgrounds
- Prepare for black background composite

### STEP 3: SCENE BUILDING
**Virtual Sets:**
- Build scenes in 3D software
- Place models in environment
- Set up virtual cameras
- Program camera movements

**Silhouette Scenes:**
- Either: Shoot actors against green screen
- Or: Use 3D character models
- Light only from behind (screens, projections)
- Keep faces in shadow
- Focus on body language

### STEP 4: LIGHTING (Virtual)
**3D Scene Lighting:**
- Set up virtual lights (sun, room lights, screens)
- Control intensity, color, shadows
- Render multiple passes if needed
- Can adjust in post

**Practical Elements:**
- Film real screens (for glow effect)
- Film real light fixtures
- Composite into 3D scenes

### STEP 5: RENDERING
- Render 3D scenes (long process)
- High quality settings
- Motion blur enabled
- Depth of field calculated
- Color grading in render or post

### STEP 6: COMPOSITING
- Layer 3D renders
- Add stock footage
- Composite silhouettes
- Add screen graphics
- Add text overlays
- Final color grade

### STEP 7: ANIMATION
- Animate text appearing
- Animate graphic elements
- Animate camera moves (if not rendered)
- Animate transitions

- Lighting changes (color, intensity, direction)
- Camera movement (reveals, approaches)
